Geometric Circle Calculator


📊 Logic Overview
This project demonstrates the implementation of geometric principles within a software environment
Constant Protection: Uses the const keyword to ensure the value of $\pi$ (3.14) cannot be modified during program execution.
User Input Handling: Dynamically accepts the radius from the user via the standard input stream.
Formula Application: Processes the mathematical expression 2 * radius * radius * pi to derive the circumference.
Precision Output: Displays the calculated result with decimal accuracy using the float type.


🛠️ FeaturesRead-Only Variables: Demonstrates best practices by using const to prevent accidental data corruption of mathematical constants.
Interactive Interface: Provides clear prompts for user interaction in the console.
